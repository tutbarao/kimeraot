Caso tenha problemas com conexão ou ping, realize um flush no DNS do seu pc.

1- Pressione as teclas Windows+R juntas para a abrir o menu “executar” e então digite cmd para abrir o console do Prompt de Comando do Windows.
2- Insira o seguinte comando para limpar os arquivos de cache DNS do seu computador:
ipconfig /flushdns
Dê enter.

Após isso seu DNS estará limpo, isso ajudará caso sua conexão não esteja boa.
